# Real-Time Messaging Platform

A full-stack real-time messaging web application built with Java Spring Boot and React.

## Tech Stack

Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT
- WebSocket
- STOMP
- MySQL

Frontend
- React.js
- React Router
- Axios

## Features

- User Registration & Login
- JWT Authentication
- Private & Group Chat
- Real-Time Messaging
- Typing Indicators
- Read Receipts
- Online/Offline Presence
- Profile Management
- File & Image Sharing
- Message Search
